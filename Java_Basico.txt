Java Básico. Java é uma linguagem de programação orientada a objetos amplamente utilizada para desenvolvimento de sistemas, aplicações web, aplicativos Android e sistemas corporativos. Variáveis são utilizadas para armazenar valores na memória. Exemplos: String nome igual a Marcos, int idade igual a 20, double altura igual a 1.75 e boolean estudante igual a true. Tipos primitivos: byte para números inteiros pequenos, short para números inteiros de tamanho médio, int para números inteiros, long para números inteiros grandes, float para números decimais de menor precisão, double para números decimais de maior precisão, char para caracteres e boolean para valores verdadeiro ou falso. Operadores matemáticos: soma, subtração, multiplicação, divisão e resto da divisão. Estruturas condicionais: if executa um bloco de código quando uma condição é verdadeira, if else permite definir uma ação para verdadeiro e outra para falso, switch é utilizado quando existem várias opções possíveis. Laços de repetição: for é utilizado quando se sabe quantas vezes uma repetição ocorrerá, while quando não se sabe exatamente quantas repetições serão necessárias e do while executa pelo menos uma vez antes de verificar a condição. Métodos são blocos de código reutilizáveis, como métodos para somar números, calcular média ou exibir mensagens. Entrada de dados pode ser realizada utilizando a classe Scanner para receber informações como nome, idade e salário. Saída de dados normalmente é realizada utilizando System.out.println para exibir informações e resultados. Boas práticas incluem utilizar nomes descritivos para variáveis, evitar código duplicado, organizar o código em métodos, manter a indentação correta e utilizar comentários apenas quando necessário. Exercícios sugeridos: criar um programa que exiba Olá Mundo, criar um programa que receba nome e idade do usuário, criar uma calculadora simples, criar um programa que mostre os números de 1 a 100 e criar um programa que informe se um número é par ou ímpar.
