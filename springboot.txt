# Spring Boot

Spring Boot é um framework para desenvolvimento de aplicações Java.

## Estrutura Básica

* Entity
* Repository
* Service
* Controller

## Entidade

```java
@Entity
public class Produto {

    @Id
    private Long id;

    private String nome;
}
```

## Repository

```java
public interface ProdutoRepository
extends JpaRepository<Produto, Long> {
}
```

## Controller

```java
@RestController
@RequestMapping("/produtos")
public class ProdutoController {
}
```

## CRUD

Create - Criar

Read - Ler

Update - Atualizar

Delete - Excluir

## Endpoint GET

```java
@GetMapping
public List<Produto> listar(){
    return repository.findAll();
}
```
