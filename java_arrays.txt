# Arrays em Java

Arrays armazenam vários valores do mesmo tipo.

## Declaração

```java
String[] nomes = new String[3];
```

## Inicialização

```java
String[] nomes = {
    "João",
    "Maria",
    "Pedro"
};
```

## Acessando posições

```java
System.out.println(nomes[0]);
```

## Percorrendo Arrays

```java
for(int i = 0; i < nomes.length; i++){
    System.out.println(nomes[i]);
}
```

## Características

* Tamanho fixo.
* Índices começam em zero.
* Todos os elementos possuem o mesmo tipo.

## Exercícios

1. Armazenar 5 números.
2. Exibir todos os números.
3. Calcular a média.
