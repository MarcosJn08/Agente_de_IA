# Programação Orientada a Objetos

POO organiza o código em objetos e classes.

## Classe

```java
public class Pessoa {

    String nome;
    int idade;

}
```

## Objeto

```java
Pessoa p = new Pessoa();
```

## Encapsulamento

Protege atributos usando private.

```java
private String nome;
```

Utilizando getters e setters.

## Herança

Permite reutilizar código.

```java
public class Animal {
}
```

```java
public class Cachorro extends Animal {
}
```

## Polimorfismo

Permite tratar objetos diferentes de forma uniforme.

```java
Animal animal = new Cachorro();
```

## Abstração

Oculta detalhes complexos e mostra apenas o necessário.

## Exercícios

1. Criar classe Pessoa.
2. Criar classe Produto.
3. Criar sistema de biblioteca usando POO.
