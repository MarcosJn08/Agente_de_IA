# SQL Básico

SQL é a linguagem utilizada para manipular bancos de dados relacionais.

## SELECT

```sql
SELECT * FROM usuarios;
```

## WHERE

```sql
SELECT * FROM usuarios
WHERE idade > 18;
```

## INSERT

```sql
INSERT INTO usuarios(nome, idade)
VALUES ('Marcos', 20);
```

## UPDATE

```sql
UPDATE usuarios
SET nome = 'João'
WHERE id = 1;
```

## DELETE

```sql
DELETE FROM usuarios
WHERE id = 1;
```

## JOIN

```sql
SELECT *
FROM pedidos p
INNER JOIN clientes c
ON p.cliente_id = c.id;
```
